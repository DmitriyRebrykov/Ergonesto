Детальний опис продукту

ERGONESTO - це преміальна ергономічна підставка для ніг, розроблена спеціально для офісних працівників, які проводять багато часу за робочим столом. Наш продукт поєднує в собі інноваційний дизайн, високоякісні матеріали та науково обґрунтовані ергономічні принципи.

Основні переваги:

  • Покращення кровообігу: Правильне положення ніг сприяє кращому кровообігу та зменшує втому
  • Зменшення болю в спині: Підтримка правильної постави знижує навантаження на хребет
  • Підвищення продуктивності: Комфортне робоче місце покращує концентрацію та ефективність
  • Універсальність: Підходить для будь-якого типу робочого столу та стільця

Матеріали та якість:

Корпус виготовлений з високоякісного натурального дерева з екологічним покриттям. М'яка подушка виконана з гіпоалергенного матеріалу з ефектом пам'яті, що забезпечує максимальний комфорт навіть при тривалому використанні.

Instructions Panel

Інструкція з використання

Розпакування

  1. Обережно витягніть підставку з коробки
  2. Перевірте комплектність (підставка, інструкція, гарантійний талон)
  3. Огляньте виріб на предмет пошкоджень
  4. При виявленні дефектів негайно зверніться до служби підтримки

Налаштування

  1. Розмістіть підставку під робочим столом
  2. Відрегулюйте висоту за допомогою механізму регулювання
  3. Встановіть комфортний кут нахилу (за потреби)
  4. Переконайтесь, що підставка стійко стоїть на підлозі

Правильне використання

Правильна позиція

Ноги повинні повністю стояти на підставці, коліна під кутом 90-110°

Періодичність

Робіть перерви кожні 30-45 хвилин, змінюйте положення ніг

Регулювання

Підбирайте висоту індивідуально під свій зріст та стіл

Догляд та обслуговування

  • Протирайте поверхню вологою тканиною щотижня
  • Використовуйте м'які миючі засоби без абразивів
  • Уникайте прямого сонячного світла та нагрівальних приладів
  • Перевіряйте кріплення раз на місяць
  • При необхідності змащуйте механізм регулювання

Важливі застереження

  • Не перевищуйте максимальне навантаження 100 кг
  • Не використовуйте як сходинку або підставку
  • Не залишайте на підставці важкі предмети
  • При пошкодженні припиніть використання

Часті питання

Так, ERGONESTO підходить для всіх стандартних робочих столів висотою від 70 до 80 см. Завдяки регульованій висоті (10-20 см) ви зможете налаштувати оптимальне положення для будь-якого стільця.

Вага підставки становить 2.5 кг. Це оптимальна вага для забезпечення стабільності та зручності переміщення.

Так, м'яка подушка з memory foam забезпечує комфорт навіть при використанні без взуття. Поверхня гіпоалергенна та приємна на дотик.

Рекомендуємо протирати поверхню вологою тканиною раз на тиждень. Для глибокого очищення можна використовувати м'які миючі засоби без абразивів.

Так, ми надаємо спеціальні умови для корпоративних замовлень від 10 штук. Зверніться до нашого менеджера для отримання індивідуального комерційного пропозиції.
